# SIOT College Application

The SIOT College Application is a professional **Android-based management system** designed for the Saraswati Institute of Technology. It serves as a comprehensive digital companion for students and staff, aiming to reduce paperwork, automate academic workflows, and centralize all critical records onto an integrated platform.

---

## Tech Stack

| Category | Component | Details |
| :--- | :--- | :--- |
| **Frontend** | **Java** | Core logic for the native Android application. |
| **UI/UX** | **XML Layouts** | Used for building the user interface. |
| **Styling** | **Material Design** | Components for a modern, clean look. |
| **Database** | **Firebase Realtime DB** | Storage for attendance, notes, student records, and staff data. |

### Supported Platforms
* **Mobile OS:** Android 6.0 (Marshmallow) and above.
* **Development:** Supported on Windows or macOS.

### Hardware Requirements (for Development)
* Minimum 8GB RAM.
* Intel i5 processor or equivalent.

## Installation

This is a **native Android Java project**.

1.  **Clone this repository**
    ```bash
    git clone [https://github.com/yourusername/SIOT-College-Application.git](https://github.com/yourusername/SIOT-College-Application.git)
    ```
2.  Open the project in **Android Studio**.
3.  Install the required tools: **Android SDK** (min Android 6.0), **JDK**, and **Gradle**.
4.  Allow Android Studio to **Sync Gradle**.
5.  Connect an Android device or start an emulator.
6.  Run the project by clicking **Run â–¶** in Android Studio. The application will be automatically installed on the device.
